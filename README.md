# 德州扑克 · 好友房

一个可以和朋友**在线联机**玩的德州扑克（Texas Hold'em）平台。纯 Node.js + WebSocket，无需数据库，开箱即跑。

> 玩法为**娱乐记分币（play money）**，不涉及任何真实货币赌博。

## 功能

- 房主创建房间 → 生成 4 位房间码 → 朋友输入房间码即可加入
- 支持 2~9 人同桌，庄家按钮自动轮转
- 完整下注流程：盲注、翻前/翻牌/转牌/河牌、弃牌/过牌/跟注/下注/加注/全下
- **边池（Side Pot）** 正确计算，全下分池、多人分池都支持
- 7 张牌最佳手牌评估（含同花顺、葫芦、轮子 A-2-3-4-5 等）
- 实时同步：每人只能看到自己的手牌，摊牌时自动亮牌
- 操作记录面板

## 运行

```bash
cd texas-holdem
npm install        # 安装 ws 依赖（首次）
npm start          # 启动服务器，默认端口 3000
```

浏览器打开 `http://localhost:3000`，输入昵称 → **创建房间**。
把显示的 4 位房间码（或"复制邀请链接"）发给朋友即可。

## 和朋友联机

### 方案 A：同一局域网（最简单）
- 启动服务器的那台电脑会显示一个 **局域网地址**，例如 `http://192.168.0.104:3000`
- 同一 Wi-Fi 下的朋友用这个地址打开即可（手机也行）

### 方案 B：跨互联网（朋友不在同一网络）
任选一种让服务器可被公网访问的方式：
1. **内网穿透**：在本机运行 `ngrok http 3000`（或 cloudflared / 花生壳），把生成的公网地址发给朋友。
2. **部署到云端**：把整个 `texas-holdem` 目录部署到任意支持 Node 的云服务器 / 容器，用公网域名访问。
3. 也可让我帮你一键部署到 CloudStudio 沙箱，获得一个公网可玩的链接。

> 端口可在启动时改：`PORT=8080 npm start`


## 目录结构

```
texas-holdem/
├── server.js        # HTTP 静态服务 + WebSocket 房间/广播
├── engine.js        # 牌局引擎（发牌/下注/边池/比牌），可单独测试
├── test.js          # 引擎自测
├── e2e.js           # WebSocket 端到端联机测试（需先 npm start）
├── package.json
└── public/          # 前端单页（index.html / style.css / client.js）
```

## 自定义

在 `engine.js` 的 `Game` 构造函数可调整：
- `smallBlind` / `bigBlind`：盲注大小
- `startingChips`：每人初始筹码
- `maxSeats`：最大座位数（默认 9）

也可在客户端"创建房间"时通过消息 `opts` 传入，欢迎按需扩展（如锦标赛模式、计时器、聊天等）。
